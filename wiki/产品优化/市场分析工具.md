# 市场分析工具

亚马逊市场分析的核心工具和使用方法，内容来源于《卖家精灵选市场指引》、《商机探测器专题汇报》和《市场分析 李国庆》。

## 一、卖家精灵

### 1.1 基本设置

市场报表的数据基于基本参数的设置，取值不同，市场报表里的数据不同。应根据自己的市场目标和需求设置取值。

### 1.2 选择类目

- **通过类目导航直达目标类目**：拿到目标市场的市场报表和深度分析图表
- **选择一个大类或不限定类目**：拿到大类下所有细分市场的报表和深度分析图表，选出适合自己进入的市场

### 1.3 站点选择

目前支持美国、日本、英国、德国、法国、意大利、西班牙七个站点

### 1.4 市场供需指标

- **供**：亚马逊前台该类目下上架的商品总数
- **需**：样本listing的月总销量（市场容量）

### 1.5 竞争难度指标

#### 市场平均指标

- **平均评论数**：越多，则前期积累评论的成本和难度越大
- **平均星级**：4星及以上的高星级市场，产品升级空间小，退换货、售后成本低
- **平均卖家数**：越多，跟卖越严重
- **亚马逊自营占比**：高的市场要注意
- **A+占比**：越高，对listing页面质量要求越高

#### 新品指标

- **新品平均评论数**：判断新品积累评论的速度
- **新品平均星级**：判断新品满意度
- **新品平均价格**：看是否有低价竞争，判断机会价格区间

#### 垄断指标

- **商品集中度**：越低，头部商品垄断越低
- **品牌集中度**：品牌数量越多，集中度越低，垄断程度越低
- **卖家集中度**：卖家数越多，集中度越低，垄断程度越低

### 1.6 新品定义

定义上架多长时间的产品为新品，有三个选择：3/6/9个月

**一般规则**：市场新品越多，机会越大（除服装等更新快的类目）

### 1.7 Listing样本数量

取类目下销量排名前100产品的数据作为样本来代表整个市场的数据，它是整个市场数据的缩影。

### 1.8 头部商品

样本Listing中，销量最高的商品。有四个选择：销量前3/5/10/20的产品，用于和整个市场的数据对比判断竞争。

### 1.9 市场数据看板

#### 行业需求及趋势

- **横坐标**：时间
- **纵坐标**：搜索量
- **作用**：了解市场需求的变化、季节性、事件性、节日性、备货准备

#### 商品集中度

- **作用**：分区间研究新品机会、商品垄断
- **判断标准**：柱状图走势越平缓，销量分布越平均，垄断程度越低

#### 品牌集中度

- **作用**：研究品牌垄断、新品牌机会
- **判断标准**：品牌数多，表明该市场的消费者没有只认准某几个品牌

#### 卖家集中度

- **作用**：研究卖家垄断程度

#### 卖家类型

- **作用**：了解卖家类型分布

#### 上架时间

- **作用**：了解产品上架时间分布

#### 上架趋势

- **作用**：了解产品上架趋势

#### 评论数分布

- **作用**：了解评论数分布情况

#### 评分值分布

- **作用**：了解评分值分布情况

#### 价格分布

- **作用**：了解价格分布情况

### 1.10 市场报表排序规则

**作用**：用报表内的任一数据维度来排列市场，高效找到适合自己进入的市场

**常用排序条件**：
- 商品/品牌/卖家集中度的升序排列
- 新品数量降序排列
- FBM发货占比降序排列

### 1.11 产品可行性分析

- **市场样本数据详情**：了解市场整体情况
- **头部商品数据详情**：了解头部商品情况
- **新品数据详情**：了解新品情况

## 二、商机探测器

### 2.1 什么是商机探测器

**官方定义**：借助该工具，卖家可以确定发布高潜力新商品的最佳时机，以便填补买家的需求空白。

**功能**：
- 借助亚马逊大数据分析和人工智能算法
- 帮助卖家挖掘细分市场搜索词
- 捕捉未满足买家需求和竞争产品的信息
- 深入分析到ASIN层面
- 节省卖家开发高价值新产品所需的时间

### 2.2 登入方式

1. 进入亚马逊账户后台
2. 点击后台【growth】的选项
3. 下方找到product opportunity explorer
4. 点击进入商品机会探测器界面

**已开放权限**：美国站、英国站和欧洲8国站点（德、法、意、西、瑞、波、比、荷）以及日本站

### 2.3 访问细分市场的方式

1. **按分类浏览细分市场**
2. **搜索细分市场**（最常用）
3. **使用示例细分市场**

### 2.4 搜索方式

#### 方式1：用关键词进行搜索

搜索目标关键词即可得出符合买家需求的多个细分市场及ASIN：

**Niche视图**：
- 按细分市场详情、搜索量、销售额、价格详情进行了归类
- 数据维度为细分市场数据

**ASIN视图**：
- 可以了解该搜索词相关的商品
- 按ASIN详情、需求、价格详情、客户情绪、市场竞争进行了归类
- 数据维度为亚马逊全站的数据
- 展示上限1000个的相关产品及其基本数据信息

#### 方式2：用ASIN进行搜索

搜索目标商品的ASIN即可得出符合买家需求的多个细分市场及ASIN：

**Niche视图**：
- 可以查看该ASIN所在的若干个细分市场的具体情况

**ASIN视图**：
- 可以查看该ASIN及其类似ASIN（上限为100个）的基本数据信息

### 2.5 Niche视图表头指标

- **热门搜索词**：近90天内该细分市场搜索量最高的TOP3词
- **点击量最多的商品数量**：基本判定为这个细分市场的活跃商品数，也即主要竞争对手数量
- **搜索量**：360天内输入当前细分市场中任意搜索词的总次数（不去重，非搜索人次）
- **售出商品数量**：可以用来评估市场容量

### 2.6 ASIN视图表头指标

- **搜索词层面**：统计的数据通常为搜索量
- **商品ASIN层面**：统计的数据通常为点击量
- **总评分和平均买家评分**：即评论数和星级
- **总报价数量**：即90天内售卖该商品的最多卖家数量

### 2.7 细分市场详细信息

**定义**：细分市场是指代表买家需求的---搜索词和商品。当买家输入搜索词时，展现了他们的购买需求。亚马逊会根据买家查看或购买的商品将这些搜索词进行相应的组合，形成"细分市场"，其中最高频的关键词是细分市场的名称。

### 2.8 商品模块分析

可以查看细分市场的竞品情况，挖掘以下信息：

#### 新品情况

对发布日期进行排序，观察对应的点击份额：
- 近段时间新品数量越多，点击份额越大
- 说明该细分市场下新品的接受度越高

#### 了解价格敏感度

- 如果平均售价较低的商品的点击份额较高，则可能表明对价格更敏感
- 如果低平均售价商品的点击份额并不高，则意味着买家会考虑链接质量、评论或其他标准来购买商品

#### 星级评论数的影响

- 如果总评分数或评分值较高的商品的点击份额较高
- 则表示买家在购买该商品时更加重视评分和评论

#### 竞品品牌分析

可通过分品牌累积汇总点击份额找出主要竞品品牌

### 2.9 搜索词模块分析

搜索词模块包含细分市场内热门买家搜索词，以及以下指标：

#### 搜索量增长情况

- 如果大多搜索词在过去180天内出现高速增长，但在过去90天内出现负增长
- 则可能表明对该产品的需求存在短期下降趋势

#### 点击份额

可以看出哪些词可以为产品带来更高的点击量

#### 搜索转化

- **定义**：客户输入此搜索词并点击商品后，该细分市场中任何商品的购买次数除以客户输入该搜索词的总次数
- **与广告转化率的区别**：搜索转化的分母为搜索量，而广告转化率的分母为点击量
- **意义**：辅助判断市场的供求关系健康度

#### TOP3产品

细分市场中点击量最高的3个产品

### 2.10 洞察数据层面

进一步了解该细分市场，评估：
- 如果在这个细分市场下发布新品，预期的潜力如何？
- 竞争如何？
- 机会如何？
- 广告推广的难易程度
- 广告效果是否容易达到

### 2.11 趋势数据层面

主要展现五个层面的数据：
1. 搜索量
2. 转化率
3. 平均价格
4. 产品数量
5. 品牌数量

## 三、ABA（Amazon Brand Analytics）

### 3.1 ABA概述

ABA工具页面有了新变化，主要包括两个板块：
- **全系商品搜索表现**：基于自己的ASIN表现排名前1000的关键词数据
- **搜索词表现**：整个大盘根据客户搜索频率进行关键词排名的数据

### 3.2 ABA核心功能

#### Top Clicked Products（点击量最高的商品）

通过输入ASIN可以反查出该ASIN在指定时间范围内点击份额进入了前三的所有关键词

#### Top Clicked Brand（点击量最高的品牌）

通过输入一个品牌名称，可以反查出该品牌在指定时间范围内所有的listing点击份额进入了前三的所有关键词

#### Search Term（关键词搜索）

通过输入一个关键词，可以查询出所有包含你输入的单词的搜索词（顺序颠倒或者中间被插入其他的关键词也会被搜索出来，错别字和单复数都无法识别）

### 3.3 ABA关键指标

#### Click Share（点击占比）

```
点击占比 = 该ASIN在这个关键词下被点击数 / 该关键词下所有的ASIN点击数
```

#### Conversion Share（转化占比）

```
转化占比 = 该ASIN在这个关键词下的出单数 / 该关键词的出单数
```

#### 转化效率判断

如果转化占比/点击占比 > 1，则代表着该ASIN的转化率高于平均转化率。

**公式推导**：
```
转化占比/点击占比 = 该ASIN在这个关键词下的出单数 * 该关键词下所有的ASIN点击数 / 该关键词的出单数 * 该ASIN在这个关键词下被点击数
= 该ASIN在这个关键词下的点击数² * 该ASIN转化率 / 该ASIN在这个关键词下被点击数² * 该关键词转化率
```

### 3.4 ABA使用建议

1. **每周下载一次ABA数据报告**
2. **整理对应关键词的搜索排名**
3. **整理Top3点击占比之和和Top3转化占比之和**

### 3.5 关键词数据波动

在实操中有些关键词在某个时间段会没有数据，这个是正常的，因为关键词频率在不断波动和变化，时高时低，甚至可能直接跌出亚马逊热搜索榜单。

**应对策略**：
- 如果大部分出单来源于这一类关键词，那么就可能出现突然的销量下跌
- 需要分析自身小类和大类的排名、自身流量词数量、进入ABA关键词数量
- 分析竞品的大类和小类排名、站内站外活动情况、广告位、自然位以及关键词搜索量
- 基于历史数据波动变化及规律可以提前判断对应词的趋势提前做准备

## 四、Google Trends

### 4.1 使用方法

1. 在谷歌趋势的搜索栏中输入产品的关键词
2. 选择对应的国家、时间标签
3. 出现搜索趋势结果
4. 可对多个词进行比较

### 4.2 全球范围查看

- 可以看到在不同国家的搜索热度情况
- 可以选择时间区间，查询特定时间内的搜索词热度情况
- 下方会给予相关词推荐，可作为广告拓词参考

### 4.3 应用场景

- 了解产品热度情况
- 判断产品生命周期
- 安排产品推广时间

## 五、竞品流量结构分析

### 5.1 关键词流量占比

**定义**：产品通过不同流量词获得的曝光量占比，也就是不同关键词为产品带来的流量大小

### 5.2 流量词分类

#### 主要流量词

- 流量占比（曝光量占比）≥ 5%
- 且自然搜索排名在第一页的前8位
- 或流量占比 ≥ 10%的流量词

#### 精准流量词

- 周曝光量 ≥ 2000
- 且自然搜索排名在第一页前8位的流量词

#### 精准长尾词

- 500 ≤ 周曝光量 < 2000
- 且自然搜索排名在第一页前8位的流量词

### 5.3 流量结构指标

- **流量占比**：不同关键词为产品带来的流量大小
- **自然流量占比**：自然搜索带来的流量占比
- **广告流量占比**：广告带来的流量占比
- **自然排名**：关键词的自然搜索排名
- **自然排名页码**：关键词在搜索结果的第几页
