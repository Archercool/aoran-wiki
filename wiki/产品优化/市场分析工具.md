# 市场分析工具

亚马逊市场分析的核心工具和使用方法。

## 卖家精灵

### 市场报表功能

- 支持美国、日本、英国、德国、法国、意大利、西班牙七个站点
- 基于基本参数设置，取值不同数据不同

### 市场供需指标

- **供**：亚马逊前台该类目下上架的商品总数
- **需**：样本listing的月总销量（市场容量）

### 竞争难度指标

- **新品数量**：新品越多，占比越大，市场机会越大
- **卖家集中度**：卖家数越多，集中度越低，垄断程度越低
- **商品集中度**：越低，头部商品垄断越低
- **品牌集中度**：品牌数量越多，集中度越低，垄断程度越低

### 市场数据看板

- 行业需求及趋势
- 商品集中度
- 品牌集中度
- 卖家集中度
- 卖家类型
- 上架时间
- 评论数分布
- 评分值分布
- 价格分布

## 商机探测器

### 登入方式

在紫鸟浏览器卖家后台主页中，点击：左上角菜单→增长→商机探测器

### 搜索方式

1. **关键词搜索**：用关键词搜索细分市场（最常用）
2. **ASIN搜索**：用ASIN搜索细分市场
3. **类目搜索**：按照类目搜索细分市场

### Niche视图表头指标

- **热门搜索词**：近90天内该细分市场搜索量最高的TOP3词
- **点击量最多的商品数量**：基本判定为该细分市场的活跃商品数
- **搜索量**：360天内输入当前细分市场中任意搜索词的总次数
- **售出商品数量**：评估市场容量

### ASIN视图表头指标

- **搜索词层面**：统计搜索量
- **商品ASIN层面**：统计点击量
- **总评分和平均买家评分**：评论数和星级
- **总报价数量**：90天内售卖该商品的最多卖家数量

## ABA（Amazon Brand Analytics）

### 核心功能

- **全系商品搜索表现**：基于自己的ASIN表现排名前1000的关键词数据
- **搜索词表现**：整个大盘根据客户搜索频率进行关键词排名的数据

### 关键指标

- **Click Share（点击占比）**：该ASIN在这个关键词下被点击数 / 该关键词下所有的ASIN点击数
- **Conversion Share（转化占比）**：该ASIN在这个关键词下的出单数 / 该关键词的出单数

### 使用建议

- 每周下载一次ABA数据报告
- 整理对应关键词的搜索排名、Top3点击占比之和和Top3转化占比之和

## Google趋势

### 使用方法

1. 在搜索栏中输入产品关键词
2. 选择对应的国家、时间标签
3. 查看搜索趋势结果
4. 可对多个词进行比较

### 应用场景

- 了解产品热度情况
- 判断产品生命周期
- 安排产品推广时间
