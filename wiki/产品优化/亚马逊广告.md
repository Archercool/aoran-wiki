# 亚马逊广告

亚马逊站内广告的核心知识和投放策略，内容来源于《广告基础系列》和《01广告说明书2023年版》。

## 一、广告目的

### 1.1 核心认识

广告就是一个工具，是锦上添花而非雪中送炭。将listing关键词写好，落地页优化好之后，再去开广告。

### 1.2 两个主要目的

1. **获取曝光和流量**：通过广告在短时间内尽可能去获得大的曝光和流量
2. **促进单量和提升转化率**：推动BS类目排名，以及关键词自然排名，研究如何稳住排名和单量，占住坑位

### 1.3 终极目的

**盈利！**运营的本质就是利润，盈利才是打广告的终极目的。

## 二、广告基础概念

### 2.1 广告分类

根据广告展示效果差异以及投放的目的区别，分为：
- **SP广告**（Sponsored Products）
- **SB广告**（Sponsored Brands）
- **SD广告**（Sponsored Display）

### 2.2 自主性分类

- **自动广告**：依赖亚马逊的机器算法进行自动投放
- **手动广告**：根据自己的需求进行选择投放

### 2.3 投放对象分类

- 关键词投放
- 商品投放
- 受众投放（SD独有）
- 视频投放

### 2.4 匹配模式分类

**针对关键词**：
- Close-match（紧密匹配）
- Loose-match（宽泛匹配）
- Broad（广泛匹配）
- Phrase（词组匹配）
- Exact（精准匹配）

**针对ASIN（商品）**：
- Substitutes（替代商品）
- Complement（互补商品）
- CT（类目投放）
- PT（商品投放）
- PT-expand（商品拓展）

**针对人群**：
- 亚马逊消费者
- 浏览再营销定向人群
- 购买再营销定向人群

### 2.5 竞价相关概念

- **bid（竞价）**：对具体广告投放对象设置的点击价格
- **bid+**：建立在已有bid设置的基础上，针对不同广告位进行额外的bid上浮设置
- **竞价模式**：针对bid设置后实际产生花费时的约束

## 三、竞价机制

### 3.1 第二位竞价模式

卖家并不需要为每次点击支付全部竞价，只需要支付排名在他们下一位的卖家的价格。

**示例**：
- 第一位卖家标价为$4，但只需要支付下一位广告客户的价格$3
- 同样适用于第二、第三个广告客户

### 3.2 广告质量得分

亚马逊会结合以下因素给广告一个分数：
- 预期点击率
- 落地页面的体验（转化率）
- 广告的相关性
- 历史表现等

**广告排名得分** = 广告出价（竞价）及广告质量得分的综合运算

### 3.3 实际点击扣费公式

```
实际点击扣费 = 下一位广告排名得分 / 你的广告质量得分 + 0.01
```

实际产品的单次点击费用是市场各方卖家博弈后的结果，非一成不变的。

## 四、SP广告详解

### 4.1 SP广告概述

**定义**：利用关键词或商品匹配用户搜索，在亚马逊搜索结果页面及商品详情页面上精准定向展示商品的广告形式。

**优势**：增加曝光度、销售机会、高关联度、高透明度

**展示位**：搜索页面顶部、中间、下方，商品详情页中间、底部

**前提条件**：拥有黄金购物车的店铺才可以投放SP广告

### 4.2 自动广告

自动广告是自主性最低的广告，无法自主选择具体的投放对象，主要是让亚马逊强大的数据库选择与产品相匹配的顾客搜索词。

**四种匹配模式**：

#### 4.2.1 Close-match（紧密匹配）

用户使用和你的商品紧密相关的搜索词进行搜索时，亚马逊会向其显示你的广告。

**特点**：
- 基于listing中的内容来匹配对应的关键词
- 可进行快速准确的测词
- 匹配的都是根据listing关键词出来的精准流量

#### 4.2.2 Loose-match（宽泛匹配）

用户使用与你的商品并不密切相关（但仍有相关性）的搜索词进行搜索时，亚马逊会显示你的广告。

**特点**：
- 对新品作用不是很大
- 适合拓展更多流量

#### 4.2.3 Substitutes（同类商品匹配）

用户查看与你的商品类似的商品详情页面时，亚马逊会向其显示你的广告。

**特点**：
- 基于商品的匹配
- 可作为定投测ASIN
- 同类产品表现较好的ASIN可以添加到手动广告里面添加为商品定投

#### 4.2.4 Complement（关联商品匹配）

用户查看与你的商品互补的商品详情页面时，亚马逊会向其显示你的广告。

**特点**：
- 除非目标性较强的产品或者关联销售性质较强的商品
- 例如：鞋子+袜子、衣服+裤子+裙子、手机+充电器
- 其他产品基本很少能得到转化

### 4.3 手动广告

手动广告是优化人员自主投放的广告，投放对象主要分为词和商品（ASIN）。

#### 4.3.1 关键词匹配模式

**Broad（宽泛匹配）**：
- 匹配时容忍度最高的模式
- 对搜索词没有特定的顺序要求
- 支持单复数、拼写错误、介词、标点符号、ing形式
- 2023年后还会匹配相关购物查询内容和与关键词近似的其他变体
- 只要客户搜索词与我们投放的词存在一定的相关性即可获得展示

**Phrase（词组匹配）**：
- 对关键词有特定的顺序要求（包含投放的固定词组）
- 支持单复数、介词、ing形式的词语

**Exact（精准匹配）**：
- 要求顾客搜索词必须和我们投放的关键词词序完全匹配
- 支持单复数、介词for、短划线-、the

**匹配相关性**：精准匹配 > 词组匹配 > 广泛匹配

**匹配范围**：广泛匹配 > 词组匹配 > 精准匹配

#### 4.3.2 商品投放匹配模式

**CT（类目投放）**：
- 将投放类目下的所有ASIN作为合集
- 在合集内所有的ASIN下均有获得展示的机会
- 为提高精准度，可对星级、品牌、价格、是否会员、儿童类产品的年龄进行限定

**PT-expand（商品拓展）**：
- 针对单个商品投放
- 能够同时展示在单件商品和与其密切相关的其他商品的广告位
- 包括替代商品、补充商品和能吸引消费者的与原始商品相关的其他商品

**PT（商品投放）**：
- 在指定单件商品下进行展示

### 4.4 SP广告竞价设置

#### 4.4.1 竞价模式选择

**Down only**：
- 广告的实际CPC < 设置的bid
- 19年以前的广告只有这种出价模式

**Up and down**：
- CPC价格随市场竞争力度动态调整
- CPC的范围在0 < CPC < 2*bid
- 公司大多数广告采用这种出价模式

**Fixed**：
- 一般在控制CPC、抢占广告位的时候使用

**基于规则**：
- 常用于活动期
- 只需要设置预算和把控最终的ROAS目标
- 自动调整bid
- 前提条件：SP广告、广告活动至少已经运行30天、最近30天内至少有30次转化、广告活动必须满足10美元的每日最低预算要求

#### 4.4.2 Bid+设置

**广告位分类**：
1. **搜索结果顶部（首页）**：
   - 搜索结果首页顶行的商品推广广告位
   - 曝光、点击、转化都不会差
   - CPC也是最高的

2. **商品页面**：
   - 商品详情页面上的商品推广广告位
   - 所有非搜索结果广告位（例如"加入购物车"页面）
   - 曝光非常多，点击和转化相对不是很好
   - CPC也相对低一些

3. **搜索结果的其余位置**：
   - 搜索结果的中间或底部的商品推广广告位
   - 搜索结果的第二页及以后的所有商品推广广告位
   - 越靠后，客户一带而过的概率会越大

**设置范围**：首页bid+可以设置0-900%之间

**计算示例**：
如果设置bid+为50%，bid为0.2：
- 首页CPC在0.2~0.6：0.2 *（1+100%）*（1+50%）
- 产品页CPC在0.2~0.45：0.2 *（1+50%）*（1+50%）

### 4.5 SP广告报表

#### 4.5.1 广告层级结构

- **Campaign（广告活动）**：1个产品可以根据需求不同，设立多个campaign
- **Adgroup**：作为不同targeting（投放——词或asin）的合集

#### 4.5.2 七张常用报表

1. **Campaign name report**：了解campaign目前的预算情况，以及biding模式

2. **Targeting report**：展示卖家投放对象的表现情况
   - 自动广告：close-match、loose-match、substitutes等总计的表现情况
   - 主要用于监控手动广告关键词或者asin的表现

3. **Advertised report**：展示投放产品的详细数据
   - 一个产品可以投放不同的广告组
   - 展示这个产品在不同广告组中的表现情况

4. **Search term report**：最重要的报表
   - 目的是搜集买家搜索词
   - 根据该报表获得高质量的关键词是最重要的优化措施

5. **Placement report**：广告展示位报表
   - 发现产品在各个广告位分布的情况
   - 对广告位置投放有指导作用

6. **Purchased product report**：产品关联广告
   - 可以看到用户浏览该产品之后去了哪里下单
   - 适用于关联产品销售指导

7. **Impression-share report**：展现当前在投的广告下的词和asin的曝光份额
   - 了解每个不同targeting下客户搜索词的市场曝光份额
   - 为后续决策做出数理依据

#### 4.5.3 报表字段说明

**统一字段**：
- impressions：曝光（出现在客户屏幕内即算1次曝光）
- clicks：点击（广告被点击的次数）
- spend：花费（广告活动的总费用）
- sales：销售额（SP广告的归因期为7天）
- orders：订单量
- units：销量（实际销售的产品数量）

**计算字段**：
- CPC（点击均价）= spend / clicks
- CTR（点击率）= clicks / impressions（一般在0.5%）
- CR（转化率）= orders / clicks（正常在2%）
- ACoS（广告成本）= spend / sales（整体目标15%）

**ACoS公式拓展**：
```
ACoS = spend / sales ≈（click * CPC）/（price * orders）= CPC /（price * CR）
```

## 五、SD广告详解

### 5.1 SD广告概述

**定义**：通过吸引亚马逊站内/站外和商品相关商品的受众来拓展业务。

**前提条件**：店铺需要品牌

**付费模式**：
- VCPM（千次可见曝光付费——广告50%画面可视且持续1秒以上）
- CPC（单次点击）

### 5.2 SD广告特点

1. **面向买家投放**：基于用户画像的广告投放（货找人模式）
2. **面向站内外投放**：不仅可以获取亚马逊内部流量，还可以获取外网流量
3. **广告位置多样化**：靠近竞对购物车，最大化抢夺竞争对手的流量和销量
4. **广告定位功能非常细分**：可以直接定投对应的asin或者细分的类目

### 5.3 SD广告优势

1. 自动抓取促销信息
2. 素材可以灵活多变
3. 直观地让买家感受星级
4. 广告位置多样化且靠近竞对购物车
5. 广告定位功能非常细分

### 5.4 SD广告报表

主要运用5张报表：
1. **Advertised report**：与SP基本一致，额外含有cost type、view_impressions以及DPV
2. **Campaign report**：主要用于查看预算情况
3. **Match target report**：了解SD的入口来自哪些产品
4. **Purchased report**：了解实际购买品情况
5. **Targeting report**：了解实际投放对象的表现

## 六、SB广告详解

### 6.1 SB广告概述

**定义**：搜索结果页面及商品详情页面呈现含有品牌logo及定制化的标题和内容的广告，提高品牌知名度。

**优势**：
- 通过关键词在搜索结果上方投放广告
- 展示多个ASIN
- 广告具有三个ASIN展位
- 可自定义广告活动图片、标题及登陆页面体验

### 6.2 SB广告格式

1. **商品集**：
   - 着陆页是新着陆页时要求至少有2种商品可供购买
   - 引流至对应商品的详情页

2. **店铺旗舰店**：
   - 旗舰店至少有4个及以上的子页面
   - 每个子页面上至少有一个可供购买的商品

3. **视频广告**：默认引流至对应商品的详情页

### 6.3 投放目的

让客户逐渐地提高对你的品牌的认知程度和信任程度

## 七、无Review的SP广告玩法

### 7.1 核心原则

新品前期没有review定位情况下转化率一定是相对来说比较弱的，所以建议是不要一开始就引入过多的流量，容易起到反效果。

### 7.2 自动广告策略

1. 一开始建议只开紧密匹配
2. 广告活动竞价策略最好选择固定竞价或者仅降低
3. 搜索结果顶部策略和商品页面的bid设置一开始先不要设置哪一方面偏高
4. 自动广告开启的时候就要先把之前调研好的流量大、购买率低的关键词精准否定

### 7.3 手动广告策略

1. 一周之后点击次数大于5，而又不产生转化的关键词，可以视情况而定直接精准否定
2. 根据类目情况调整否定策略

## 八、自动广告的收录和运用逻辑

### 8.1 收录原理

自动广告不需要设置任何关键词，但是却可以通过后台匹配的方式进行匹配，主要是通过亚马逊A9算法对标题、五点、长描述A+等等进行收录关键词之后自然匹配的结果。

### 8.2 标题权重

标题的可收录性最大，远远大于其他地方填写的关键词权重。正常情况下，亚马逊listing的收录顺序是由上到下，由左到右的，因此标题中越前面的关键词也就越容易被自动广告跑出来。

### 8.3 ASIN匹配

根据所选择的自动广告匹配类型的种类，亚马逊会根据收录情况的不同进行自动选择投放。

**判断标准**：
- 如果close跑出来的关键词非常精准，且ASIN较少，说明埋词较精准
- 如果跑出来的ASIN数量较多，且都是类似的相关产品，如果一直表现不好可以否定那一类型的相关词汇
