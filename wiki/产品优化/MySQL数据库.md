# MySQL数据库

MySQL数据库的基础知识和常用操作。

## 基本概念

### 什么是MySQL

- MySQL是世界上最受欢迎的数据库管理系统之一
- 开源关系型数据库管理系统
- 使用SQL（结构化查询语言）进行数据操作

### 核心特点

- 性能高
- 可靠性强
- 易于使用
- 开源免费

## 基本操作

### 数据检索

- SELECT语句：从数据库中检索数据
- WHERE子句：过滤记录
- ORDER BY子句：排序结果
- LIMIT子句：限制返回的记录数

### 数据操作

- INSERT：插入新记录
- UPDATE：更新现有记录
- DELETE：删除记录

### 表操作

- CREATE TABLE：创建新表
- ALTER TABLE：修改表结构
- DROP TABLE：删除表

## 高级功能

### 联结（JOIN）

- INNER JOIN：返回两个表中匹配的记录
- LEFT JOIN：返回左表的所有记录和右表中匹配的记录
- RIGHT JOIN：返回右表的所有记录和左表中匹配的记录

### 子查询

- 嵌套在其他查询中的查询
- 可用于WHERE子句、FROM子句等

### 存储过程

- 预编译的SQL代码集合
- 可提高性能和代码复用性

### 触发器

- 在特定事件发生时自动执行的程序
- 可用于数据验证、审计等

### 游标

- 用于逐行处理查询结果集
- 适用于需要逐行处理的复杂操作

## 最佳实践

1. 使用适当的索引提高查询性能
2. 避免使用SELECT *，只选择需要的列
3. 使用参数化查询防止SQL注入
4. 定期备份数据库
5. 优化表结构和查询语句
