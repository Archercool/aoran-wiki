# Metabase BI工具

Metabase商业智能工具的使用指南，内容来源于《metabase基础功能使用.docx》。

## 一、Metabase概述

### 1.1 什么是Metabase

Metabase是一个开源的商业智能（BI）工具，允许用户通过简单的界面创建和共享数据可视化和仪表板。它支持多种数据库连接，包括MySQL、PostgreSQL、MongoDB等。

### 1.2 核心功能

- **数据连接**：支持多种数据库
- **查询构建**：可视化查询构建器
- **数据可视化**：图表、表格、地图等多种展示方式
- **仪表板**：将多个问题组合成仪表板
- **共享**：通过链接、嵌入或API分享结果
- **定时邮件**：定期发送报表邮件

## 二、基础操作

### 2.1 创建问题

#### 问题类型

1. **简单问题**：基础查询，选择表和字段
2. **自定义问题**：更灵活的查询，可添加筛选、汇总等
3. **原生查询**：直接编写SQL

#### 创建步骤

1. 点击"新建问题"
2. 选择数据源（表或保存的问题）
3. 选择可视化类型
4. 配置筛选条件（可选）
5. 配置汇总方式（可选）
6. 保存问题

### 2.2 可视化类型

#### 常用图表

- **表格**：最基础的数据展示
- **折线图**：展示趋势变化
- **柱状图**：比较不同类别的数据
- **饼图**：展示占比关系
- **散点图**：展示两个变量的关系
- **地图**：地理数据可视化

#### 图表配置

- **X轴**：通常为维度（如日期、类别）
- **Y轴**：通常为度量（如数量、金额）
- **分组**：按某个字段分组
- **筛选**：添加筛选条件
- **排序**：按某个字段排序

### 2.3 筛选条件

#### 筛选类型

- **等于**：精确匹配
- **不等于**：排除某个值
- **大于/小于**：数值比较
- **包含**：模糊匹配
- **在...之间**：范围筛选
- **为空/不为空**：NULL值处理

#### 时间筛选

- **相对日期**：最近7天、本月、本季度等
- **绝对日期**：指定具体日期范围
- **日期字段**：选择日期字段进行筛选

### 2.4 汇总方式

#### 聚合函数

- **计数**：COUNT
- **求和**：SUM
- **平均值**：AVG
- **最大值**：MAX
- **最小值**：MIN
- **去重计数**：COUNT DISTINCT

#### 分组汇总

1. 选择要分组的字段
2. 选择聚合函数
3. 选择要聚合的字段

## 三、仪表板

### 3.1 创建仪表板

1. 点击"新建仪表板"
2. 输入仪表板名称
3. 添加已保存的问题
4. 调整布局和大小

### 3.2 仪表板元素

#### 问题卡片

- 可调整大小
- 可添加筛选器
- 可设置刷新频率

#### 筛选器

- **时间筛选器**：按日期筛选整个仪表板
- **下拉筛选器**：从预定义选项中选择
- **输入筛选器**：自由输入筛选值

### 3.3 仪表板操作

- **全屏查看**：隐藏界面元素
- **刷新数据**：手动刷新所有数据
- **分享**：生成分享链接
- **导出**：导出为PDF或CSV

## 四、数据连接

### 4.1 支持的数据库

- MySQL
- PostgreSQL
- MongoDB
- SQL Server
- SQLite
- Oracle
- Google BigQuery

### 4.2 连接配置

#### 基本配置

- **数据库类型**：选择数据库类型
- **主机**：数据库服务器地址
- **端口**：数据库端口
- **数据库名**：数据库名称

- **SSL**：是否使用SSL连接

- **时区**：设置数据库时区

## 五、SQL查询

### 5.1 原生查询

```sql
-- 基本查询
SELECT * FROM table_name;

-- 条件查询
SELECT * FROM table_name WHERE condition;

-- 聚合查询
SELECT column_name, COUNT(*) as count
FROM table_name
GROUP BY column_name;

-- 连接查询
SELECT t1.column1, t2.column2
FROM table1 t1
INNER JOIN table2 t2 ON t1.id = t2.table1_id;
```

### 5.2 变量

```sql
-- 使用变量
SELECT * FROM table_name WHERE date >= {{start_date}} AND date <= {{end_date}};

-- 变量类型
-- Date
-- Number
-- Text
-- Boolean
-- Field Filter
```

### 5.3 模板片段

```sql
-- 条件片段
{% raw %}
{% if condition %}
AND column_name = 'value'
{% endif %}
{% endraw %}
```

## 六、权限管理

### 6.1 用户组

- **管理员**：完全访问权限
- **数据组**：数据访问权限
- **普通用户**：基本访问权限

### 6.2 权限类型

- **数据库权限**：访问数据库的权限
- **表权限**：访问特定表的权限
- **字段权限**：访问特定字段的权限
- **问题权限**：查看特定问题的权限
- **仪表板权限**：查看特定仪表板的权限

### 6.3 权限设置

1. 点击"设置" → "权限"
2. 选择用户组
3. 设置数据库权限
4. 设置表权限
5. 保存设置

## 七、定时邮件

### 7.1 创建定时任务

1. 打开问题或仪表板
2. 点击"分享" → "定时邮件"
3. 设置邮件频率
4. 添加收件人
5. 保存任务

### 7.2 邮件频率

- **每小时**
- **每天**
- **每周**
- **每月**
- **自定义**

### 7.3 邮件内容

- **附件**：PDF或CSV格式
- **正文**：可添加自定义文本
- **链接**：包含问题或仪表板链接

## 八、高级功能

### 8.1 自定义字段

```sql

SELECT *,
       column1 + column2 AS total,
       column1 / column2 AS ratio
FROM table_name;
```

### 8.2 数据规范化

- **百分比**：将数值转换为百分比
- **累计**：计算累计值
- **移动平均**：计算移动平均值

### 8.3 多序列图表

- 在同一图表中展示多个数据序列
- 支持组合图表（如柱状图+折线图）
